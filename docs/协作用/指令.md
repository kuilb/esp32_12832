# ST7567 LCD 芯片并口驱动简单手册

### 启用 8080 串口
设置 PSB 引脚为 1 (启用并口)  
设置 C86 引脚为 0 (选择 8080 模式)  

### 并行数据传输类型定义
所有操作要求 CSB 引脚拉低  
A0 引脚用于定义命令/数据 (1：数据 0：命令)  
RD / WR 引脚高电平有效，不可都使能  
| A0 | RD | WR | 描述                |
|:---|:---|:---|:--------------------|
| 1  | 0  | 1  | 读取显示数据        |
| 1  | 1  | 0  | 写入显示数据        |
| 0  | 0  | 1  | 内部状态读取        |
| 0  | 1  | 0  | 写入内部寄存器/指令 |

### 虚读取定义
DDRAM 目标地址的实际数据需要一个时钟周期来传输到总线锁存器  
即需要丢弃第一个读取的字节，从第二个字节开始读取递增地址的数据  

### DDRAM 结构
DDRAM 大小为 132列X65行 比特  
需要两个地址用于寻址：  
列地址：直接对应列像素编号 (0-131)，每执行一次数据访问，列寄存器数值自增。  
页地址：0 - 7 页用于 0 - 63 像素，每页 8 像素。  

说人话就是每次发送 8 比特数据，用于垂直刷新一列像素在一页的内容，继续发送则刷新下一列像素的内容。  

### 镜像和反转寄存器
MX (Column Scan Direction): 改变列映射 (左右翻转)  
MY (COM Scan Direction): 改变行映射 (上下翻转)。  

如果更改了 MX 设置，由于硬件映射逻辑改变，通常需要重新将数据写入 DDRAM 才能保证显示内容正确。

### 行地址电路 (用于滚动)
通过“设置显示起始行 (Display Start Line Set)”指令进行更改。  
该电路指定 DDRAM 中的哪一行作为屏幕显示的第一行 (COM0)。  
通过不断地、重复地设置起始行地址，ST7567 可以在不移动 DDRAM 中任何数据的情况下，实现平滑的屏幕滚动效果。

#### 显示开 / 关
在完整的初始化序列最后，通常才发送 0xAF (Display ON)，以确保用户看不到初始化过程中杂乱的数据。  
当设备进入休眠时，务必将 D 设为 0。这不仅能关闭显示，还能防止直流电荷残留在液晶层中，延长屏幕寿命。

#### 设置起始行
该指令指定 DDRAM 中的哪一行对应 LCD 面板顶部的第一行（COM0）。  
二进制码：范围：0x40 到 0x7F  
用于滚动。

#### SEG 扫描方向
MX = 0 正常方向，DDRAM 列地址 0 对应物理引脚 SEG0。数据从左向右排列。  
MX = 1 反向方向，DDRAM 列地址 0 对应物理引脚 SEG131。实现水平镜像翻转。

#### 反显设置
二进制码：1010011 + INV  
INV = 0 正常显示  
INV = 1 反向显示

#### 偏置比选择
这条指令用于设置 **液晶偏置比 (LCD Bias Ratio)**。它是调节液晶显示效果、对比度以及视角清晰度的核心电气参数之一。  
偏置比（Bias）决定了驱动液晶时“选择”电压与“非选择”电压之间的比例关系。正确的偏置设置能确保点亮的像素足够黑，而未点亮的像素保持透明。

* **指令格式：**
    * $A0 = 0$ (指令)
    * 二进制码：`1010001` + `BS`
    * **十六进制：** `0xA2` (BS=0) 或 `0xA3` (BS=1)

偏置比与占空比 (Duty) 对应表

ST7567 会根据你的屏幕驱动行数（Duty）和此指令的设置来调整电压分配：

| 占空比 (Duty) | BS = 0 | BS = 1 |
| :--- | :--- | :--- |
| **1/65** (常用) | **1/9 Bias** | **1/7 Bias** |
| 1/49 | 1/8 Bias | 1/6 Bias |
| 1/33 | 1/6 Bias | 1/5 Bias |
| 1/55 | 1/8 Bias | 1/6 Bias |

---

偏置电压参考 (以 1/65 Duty, 1/9 Bias 为例)

当设置方案为 **1/9 Bias** 时，内部电路会将 $V_0$ 驱动电压划分为 9 个等级，并分配给不同的引脚：

| 符号 | 定义 | 理论计算公式 |
| :--- | :--- | :--- |
| **V0** | 最高驱动电压 | $V_0$ |
| **VG** | 段驱动高压 | $\frac{2}{9} \times V_0$ |
| **VM** | 段驱动低压 | $\frac{1}{9} \times V_0$ |
| **VSS** | 系统地电位 | $0V$ |

#### 读-改-写模式
读改写模式允许你在不重新设置坐标的情况下，读取并修改同一个位置的数据。  
即读取数据操作不再自动自增列地址。  

---

#### 软件复位
软件复位和硬件引脚复位有点不同  
推荐上电时使用硬件复位  

---

#### COM 扫描方向
MY = 1 时反向方向。显存的第一行对应屏幕底部的 COM63 引脚，实现垂直镜像翻转。  

---

#### 电源控制
由三个模块组成，它们协同工作来产生液晶所需的驱动电压：  
VB：内部升压电路，将输入的 $VDD$ 电压升高到 $V_{OUT}$ 级别。这是驱动液晶的高压源。  
VR：内部电压调节器，通过内部电阻分压，将高压调节为稳定的驱动电压 $V_0$。  
VF：内部电压跟随器，提供足够的电流驱动能力，确保在点亮大量像素时电压不塌陷。  

通常情况下，这 3 个标志位是同时开启 (0X2F) 的。

---

#### 内部电压调节电阻比
与 EV (电子电量/对比度) 指令共同决定了液晶驱动电压 $V_0$ 的大小。  
二进制码：00100 + RR2 + RR1 + RR0  
十六进制范围： 0x20 (3.0) 到 0x27 (6.5)，步进 0.5  

---

#### 内部电压设置
精调对比度用  
EV[5:0]	0x00 ~ 0x3F，共 64 级。数值越大，$V_0$ 电压越高，屏幕越黑。
推荐值：0x1F (31)  

---

#### 省电模式
依次发送两条指令：  
Display OFF (0xAE)：关闭显示。  
All Pixel ON (0xA5)：将所有引脚拉低至 VSS。  

退出则是：  
Cancel All Pixel ON (0xA4)：恢复正常显示逻辑（不点亮所有像素）。  
Display ON (0xAF)：重新开启显示和内部电源电路。  

---

当供电电压极低时需要修改以维持对比度 (< 2.6V)
