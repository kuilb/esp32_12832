# 简单的 ST7567 流程
## 请参考指令手册

### 上电，执行硬复位
在上电稳定后，必须执行一次有效的硬件复位。  
**上电稳定**： 等待 $VDD1$ 和 $VDD2$ 达到额定电压的 90%（通常 $t > 1ms$）。  
**拉低 RSTB**： 保持低电平时间 $t_{RW}$（建议 $> 5\mu s$）。拉高 RSTB： 释放复位信号。  
**稳定等待**： 复位完成后等待 $t_R$（建议 $> 5\mu s$），此时芯片才开始接收指令。

---

### 初始化配置
手册建议在复位释放后的 5ms 内完成这些设置，以防止液晶受到不稳定的直流电压驱动。  
**(11) Bias Select**： 设置偏置比（如 0xA2）。  
**(8) SEG Direction & (15) COM Direction**： 确定屏幕的镜像和翻转状态（如 0xA0, 0xC8）。  
**清屏操作**： 初始化 DDRAM (Page 0~8)。

---

### 启动电源与显示
**(16) Power Control**： 发送 0x2F 启动内部升压、调节和跟随电路。  
**(17) Regulation Ratio & (18) Set EV**： 设置对比度的粗调和精调值，确保驱动电压 $V_0$ 稳定。  
**(1) Display ON**： 发送 0xAF。

---

### 写入显示数据
**Display Start Line Set (0x40~0x7F)**： 决定屏幕顶行对应 DDRAM 的哪一行。通常设为 0。如果你想实现垂直滚动效果，可以改变这个值。  
**Page Address Set (0xB0~0xB8)**： 决定操作哪一“页”（每一页高 8 像素）。  
**Column Address Set (0x00~0x1F)**： 决定操作哪一列。  
**Display Data Write**： 发送数据字节（此时 $A0=1$）。每写一个字节，列地址指针通常会自动 $+1$。  

---

### 简单的时序描述
系统周期时间：240ns  
写脉冲宽度：80ns  
读脉冲宽度： 140ns  
数据建立时间：40ns  
数据保持时间：20ns  

硬复位最小拉低电平时间：1.0 $\mu s$  
内部复位完成时间：1.0 $\mu s$
